NASA Lunabotics Rover

During my senior academic year, I worked with my fellow team members, faculty, and local industry to design and build a robot that would compete in the NASA Lunabotics Competition at Kennedy Space Center in Florida. My technical assignment on the team was to design the communication protocol and controls for the robot using an Artix-7 FPGA board and VHDL.

FPGA Atari Breakout Style Game

Using Verilog and the VGA output on an FPGA, I was able to create a fully functional game with animations, moveable objects, and a scoreboard. The game operated at a resolution of 800 by 600 pixels with a refresh rate of 72 Hz, and thus can be played on almost any monitor or television. See the code and a demonstration below!
